Ninja's Music Player

Create a sleek and dynamic "Ninja's Music Player" web application. Elevate your music experience with vibrant visuals, smooth controls, and seamless track navigation.

Welcome, developers! You are tasked with enhancing the existing foundation of the "Ninja's Music Player" web application to make it a dynamic and user-friendly music player. Leveraging the supplied HTML, CSS, and JavaScript code (starter kit), you are to enhance the visual and interactive elements of the application, providing users with a sleek and modern interface for enjoying their favourite tracks.

General requirements

Your solution should retain the structural integrity of the existing starter kit while introducing functional enhancements and visual refinements to elevate the user experience.



Task for students

  • Music player interface

    • Utilize appealing visuals and animations to create a lively and dynamic interface for the music player.
    • Ensure a smooth transition between different tracks with the forward and backward skip buttons.

  • Track details

    • Display the current track's title prominently, along with the artist's name and the track cover image.
    • Update the track details dynamically as the tracks change.

  • Progress Bar

    • Implement a functioning progress bar that accurately displays the current progress of the track being played.
    • Allow users to click on the progress bar to seek specific parts of a track.

  • Playback controls

    • Include play, skip forward, and skip backward buttons for controlling the playback of tracks.
    • The play button should toggle between play and pause states, displaying the respective icons for each state.

  • Time display

    • Show the current time and the total duration of the track, updating dynamically as the track plays.
