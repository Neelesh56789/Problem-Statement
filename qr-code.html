Ninja's QR Code Generator

Create a user-friendly QR Code Generator in ReactJS, effortlessly transforming URLs into downloadable QR codes


You have laid the foundation for an intuitive QR Code generation web application. This platform lets users input URLs, transforming them into scannable QR codes. But, the journey doesn't end here. Enhance the application to ensure it is the epitome of user-friendliness and functionality.
You have to use ReactJs with qr-code library to build this application. Your application should generate QR-Code at one click for any url. You should be able to download it and clear it, in case you don’t want to use it.


Tasks for Students:

Core Requirements

  • URL Input
  • Ensure that users can effortlessly input URLs for QR code generation.
  • If you don’t input any url and click on the Generate button, it should display an error message for 5 seconds.
  • If you put the wrong url, it should generate a falsy qr-code.
  • QR Code Generation
  • Refine the "Generate" button to produce QR codes efficiently.
  • Incorporate real-time feedback to notify users of the generation process.
  • Clear and Reset
  • Ensure the clear functionality wipes out the current QR Code and resets the input field.
  • Download Functionality
  • Allow users to download the generated QR Code in a high-resolution PNG format.
  • Alert System
  • Enhance the alert system to display meaningful error messages and notifications based on user actions.
