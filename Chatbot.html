Ninja's Chatbot

Create an interactive and visually appealing chatbot, "Ninja's Chatbot", that engages users in friendly conversations, with smooth animations and dynamic chatbot design.

Welcome, developers! You are tasked with creating a lively and interactive chatbot interface named "Ninja's Chatbot." This chatbot should be a ready assistant popping up from a web page, eager to engage users in a friendly chat. Your mission is to build an interface that is not just functional but also visually appealing and user-friendly. Use the supplied html, css, and javaScript (starter kit) code as your starting point and guide to building this interactive chatbot.

General requirements

Your solution should leverage the existing starter kit structure to create a chatbot that aligns with the following specifications and mirrors the initial user interface design portrayed in the sample codes.

How to zip & upload your project

  • You must start the project by first downloading the starter kit file attached to this project & make changes in those files only. You should not start from scratch by creating your own files.
  • Use the given starter kit file and do not make any changes in the package.json and the already provided versions of the dependencies.
  • Delete your package.lock.json and node modules before submitting the zip of the project.
  • Project folder name should not contain characters like +, (, / etc.
  • Project must be in a root folder which must be zipped and then submitted for evaluation.
