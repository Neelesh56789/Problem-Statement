Crypto Tracker

Develop a web-based crypto tracker app that allows users to monitor the real-time prices of various cryptocurrencies.

You are tasked with developing an app that should provide users with the ability to select a cryptocurrency from a list, view its current price in US dollars, and set price alerts. When a selected cryptocurrency's price reaches or surpasses a user-set alert price, the app should notify the user.

General requirements

Your solution should retain the structural integrity of the starter kit while introducing functional enhancements and visual refinements to elevate the user experience.



Task for students

  • User interface

    • The app should have a clean and user-friendly interface with a title "Crypto Tracker" prominently displayed.
    • A dropdown select element labeled "Select a cryptocurrency" should be provided to choose from a list of cryptocurrencies.
    • Display the current price of the selected cryptocurrency in US dollars next to the dropdown.
    • Include an input field for users to set an alert price.
    • Show an alert message when the cryptocurrency's price exceeds or equals the user-set alert price.

  • Styling and responsiveness

    • Apply styles to the app by creating beautiful css to make it visually appealing.
    • Ensure that the app is responsive and functions well on both desktop and mobile devices

  • Crypto data fetching

    • Implement JavaScript code to fetch a list of cryptocurrencies from the 'https://api.coinlore.net/api/tickers/' api.
    • Populate the dropdown select element with cryptocurrency options from the API response.
    • Use the 'https://api.coinlore.net/api/ticker/?id=' api to fetch the real-time price of the selected cryptocurrency when a user makes a selection.

  • Price alert

    • Enable users to input an alert price for the selected cryptocurrency.
    • Continuously monitor the cryptocurrency's price and trigger an alert message when it reaches or surpasses the user-set alert price.

  • Error handling

    • Implement error handling for api requests, displaying appropriate error messages to users if data cannot be retrieved.
