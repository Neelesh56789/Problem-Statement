Ninja Music Player Web App

Create a sleek and dynamic Ninja Music Player web app. Elevate your music experience with vibrant visuals, smooth controls, and seamless track navigation.


Welcome, developers! You are tasked with enhancing the existing foundation of the "Ninja Music Player" web application to make it a dynamic and user-friendly music player. Leveraging the supplied HTML, CSS, and JavaScript code (Scaffold), you are to enhance the visual and interactive elements of the application, providing users with a sleek and modern interface for enjoying their favourite tracks.


Tasks for Students:

Core Requirements

  • Music Player Interface
  • Utilize appealing visuals and animations to create a lively and dynamic interface for the music player.
  • Ensure a smooth transition between different tracks with the forward and backward skip buttons.
  • Track Details
  • Display the current track's title prominently, along with the artist's name and the track cover image.
  • Update the track details dynamically as the tracks change.
  • Progress Bar
  • Include play, skip forward, and skip backward buttons for controlling the playback of tracks.
  • The play button should toggle between play and pause states, displaying the respective icons for each state.
  • Time Display
  • Show the current time and the total duration of the track, updating dynamically as the track plays.
  • Use the following details in the same order.
