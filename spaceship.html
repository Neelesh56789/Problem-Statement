Spaceship Invasion

Create 'Spaceship Invasion,' a thrilling web game. Protect your territory by shooting invaders and earning points. Aim for the highest score in this exciting challenge!


You are a student developer tasked with creating an engaging game called "Spaceship Invasion" using the given HTML5 Canvas and JavaScript (Scaffold). Spaceship Invasion is an interactive game in which the player has to fire the bullet before any invader reaches to the ground. The game should feature player controls, shooting mechanics, invaders movements, scoring, and a time limit as mentioned in the Game Mechanics below.


Tasks for Students:

Core Requirements

  • Setting Up
  • Start with a basic index.html structure.
  • Build a start button that is used to start the game.
  • There should be multiple invaders, but should be one spaceship gun.
  • Initial Display
  • Build a canvas for playing the game. The game should run inside the canvas.
  • Design the movement of the player and enemy. Fix the enemies movement and give the control of the player to the user.
  • Ensure a smooth and responsive gaming experience across various devices and screen sizes.
  • Incorporate audio effects to enhance the overall gaming atmosphere.
  • Game Mechanics
  • Start the game when the 'Start Game' button is clicked.
  • Allow the player to move and shoot using keyboard controls.
  • Player can move to left or right using ‘left’ and ‘right’ arrow keys on the keyboard.
  • Player can fire the bullet using the ‘space’ key on the keyboard.
  • There should be a bullet sound when a bullet collides with an invader.
  • There should be a point losing sound when any invader reaches to the ground and the spaceship gun is not able to destroy it.
  • Implement enemy AI to create an invasion of enemies.
  • Scoring System: There should be a scoring system and it will start updating fast as a player gets the power-ups.
  • Add the timer: Incorporate a timer to limit the game's duration.
  • Game Outcomes
  • There will be no losing or winning conditions. Your aim should be to get the highest possible score.
