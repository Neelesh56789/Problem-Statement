Spaceship Invasion

Create 'Spaceship Invasion,' a thrilling web game. Protect your territory by shooting invaders and earning points. Aim for the highest score in this exciting challenge!

Welcome developer! You are tasked with creating an engaging game called "Spaceship Invasion" using the given HTML5 Canvas, CSS and JavaScript (starter kit). Spaceship Invasion is an interactive game in which the player has to fire the bullet before any invader reaches to the ground. The game should feature player controls, shooting mechanics, invaders movements, scoring, and a time limit as mentioned in the game mechanics below.

General requirements

Develop a “Spaceship Invasion” web game and protect your territory.Your game needs to follow the guidelines below and align with the provided sample video for the best UI resemblance.



Task for students

  • Setting up

    • Start with a basic index.html structure.
    • Build a start button that is used to start the game.
    • There should be multiple invaders, but should be one spaceship gun.

  • Initial display

    • Build a canvas for playing the game. The game should run inside the canvas.
    • Design the movement of the player and enemy. Fix the enemies movement and give the control of the player to the user.
    • Ensure a smooth and responsive gaming experience across various devices and screen sizes.
    • Incorporate audio effects to enhance the overall gaming atmosphere.

  • Error handling

    • Start the game when the 'Start Game' button is clicked.
    • Allow the player to move and shoot using keyboard controls.
    • Player can move to left or right using ‘left’ and ‘right’ arrow keys on the keyboard.
    • Player can fire the bullet using the ‘space’ key on the keyboard.
    • There should be a bullet sound when a bullet collides with an invader.
    • There should be a point losing sound when any invader reaches to the ground and the spaceship gun is not able to destroy it.
    • Implement enemy AI to create an invasion of enemies.
    • Scoring system: There should be a scoring system and it will start updating fast as a player gets the power-ups.
    • Update the player's score by 10 whenever any bullet collides with the invader.
    • Decrease the player’s score by 10 whenever any invader reaches to the ground.
    • There should be a highest score functionality in a single session of the game.
    • When a bullet collides with the invader and the score is equal to the highest score, then the highest score gets added by 10 additional points.
    • When bullet collides with the invader and score is less than highest score, then the highest score will remain the same.
    • Whenever the timer ends or the score reaches zero, the game should be ended.
    • At the end of the game, you have to print the highest score attained.
    • Add the timer: Incorporate a timer to limit the game's duration.
    • Set the timer for 2 minutes, otherwise you might face difficulty in test case pass
    • The game should end either the timer will reach to 0 sec or score reaches to 0.

  • Game outcomes

    • There will be no losing or winning conditions. Your aim should be to get the highest possible score.
