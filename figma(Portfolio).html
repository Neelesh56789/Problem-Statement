E-Portfolio

Improve your chances of getting placed by showcasing your E-Portfolio.

You have successfully developed a reactjs-based portfolio web application to showcase your skills and experience. However, the journey doesn't end here. The objective is to further improve the user experience and functionality of the application, making it an exceptional representation of your talent as a developer.

General requirements

You have to make this portfolio web app by using react-js and tailwind css. Your project should have all the sections that a portfolio website needs.



Task for students

  • User interface refinement

    • Elevate the user interface design to be adaptive and compatible with various devices. Consider incorporating a fresh, modern color palette to captivate users and leave a memorable impression.

  • Content Review

    • Carefully review and update the content throughout your portfolio. Ensure that your introduction effectively communicates your skills, experience, and goals, presented in a clear and concise manner.

  • Responsive design

    • Guarantee that your portfolio is fully responsive, providing an optimal user experience across different screen sizes. Test and optimize the layout and content for mobile devices.

  • Portfolio showcase enhancement

    • In the portfolio section, consider adding detailed descriptions, project specifications, and links to live projects or GitHub repositories. Visual elements such as images or demos can effectively showcase your work.

  • Contact form enhancement

    • The contact form should offer a seamless and secure user experience. Ensure that user inputs are validated and sanitized, and provide clear feedback upon successful submission.

  • Dark mode toggle

    • Introduce a dark mode toggle feature that allows users to switch between light and dark themes for improved readability and comfort.

  • Interactive elements

    • Implement interactive elements such as hover effects, transitions, or animations to engage users and create a visually appealing experience.

  • Social media integration

    • Add links to your social media profiles in the footer section, allowing visitors to connect with you on various platforms.

  • Resume download

    • Offer users the option to download your resume directly from the portfolio.

Do’s & don’t

  • You must start the project by first downloading the starter kit attached to this project & make changes in those files only. You should not start creating your own files.
  • Use the given starter kit and do not make any changes in the package.json and the already provided versions of the dependencies.
  • Delete your package.lock.json and node modules before submitting the zip of the project.
  • Project folder name should not contain characters like +, (, / etc.
  • Project must be in a root folder which must be zipped and then submitted for evaluation.
