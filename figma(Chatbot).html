Ninja's Chatbot

Create an interactive and visually appealing chatbot, "Ninja's Chatbot", that engages users in friendly conversations, with smooth animations and dynamic chatbot design

Welcome, developers! You are tasked with creating a lively and interactive chatbot interface named "Ninja's Chatbot." This chatbot should be a ready assistant popping up from a web page, eager to engage users in a friendly chat. Your mission is to build an interface that is not just functional but also visually appealing and user-friendly. Use the supplied html, css, and javaScript (starter kit) code as your starting point and guide to building this interactive chatbot.

General requirements

Your solution should leverage the existing starter kit structure to create a chatbot that aligns with the following specifications and mirrors the initial user interface design portrayed in the sample codes.



Task for students

  • Chatbot interface

    • Incorporate a chatbot toggle button that switches between an open and close state, showcasing respective icons for each state.
    • The chatbot window should smoothly transition in and out of view with appealing animations during toggling.

  • Chatbox

    • Create a chatbox that lists the chat in a visually distinguishable manner between incoming and outgoing messages.
    • The incoming chat bubbles should have different styles compared to the outgoing ones.
    • Ensure that chat messages appear with a typewriter effect to enrich the interaction experience.

  • Chat input area

    • Implement a chat input area where users can type and send messages.
    • The chat input area should grow and shrink dynamically based on the amount of text entered.
    • Integrate a "send" button that facilitates the sending of messages.

  • Chatbot Functionality

    • Leverage OpenAI's GPT-3.5-turbo to generate chatbot responses.
    • The chatbot should be able to handle errors gracefully, informing the user if something goes wrong.
    • Ensure that the chatbot displays a "thinking" message before showcasing the generated response, enhancing the real-time chat experience.
