Weather Forecast Web App


Create an intuitive Weather Forecast App, offering dynamic city-specific weather data with enhanced visuals, seamless interactions, and responsive design.


Welcome, developers! You are tasked with enhancing the existing foundation of the “Weather Forecast” web application to make it a dynamic and user-friendly weather forecasting tool. Leveraging the supplied HTML, CSS, and JavaScript code (Scaffold), you are to enhance the visual and interactive elements of the application, providing users with an intuitive and modern interface for checking the weather in different cities worldwide.


General Requirements

Your solution should retain the structural integrity of the Scaffold while introducing functional enhancements and visual refinements to elevate the user experience.

Tasks for Students

Core Requirements
  • Weather Forecast Interface
  • Utilize appealing visuals and animations to create a lively and dynamic interface for the weather forecast app.
  • Ensure a smooth user experience that facilitates easy access to weather data.
  • City and Weather Details
  • Display the current city’s name prominently, your api will generate the data of that city.
  • API will provide the weather details dynamically based on the city entered by the user (see below on how to generate API)
  • Error Handling
  • Implement effective error handling to manage incorrect city inputs or issues during the data fetch process.
  • Form Enhancement
  • You need to maintain the existing HTML structure for the interface, while being open to necessary structural enhancements.
  • Clear the form input after successfully fetching the weather data to allow for easy subsequent searches.
